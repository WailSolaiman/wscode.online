# WS Code-Online

A simple, modern one-page website for WS Code-Online.

## Tech Stack

- React 18
- Vite
- Plain CSS (no frameworks)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Responsive design
- Dark modern theme
- Smooth scrolling navigation
- Email copy functionality with toast notification
- SEO optimized
